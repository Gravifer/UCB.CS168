# CS 168: Internet Architecture and Protocols

This repository contains coursework for **UC Berkeley's CS 168 course** on Internet Architecture and Protocols. The course covers fundamental networking concepts, protocols, and implementations.

## Projects

### [Project 1: Traceroute](proj1/)
Implementation of the **traceroute** utility, which discovers the path packets take through a network. This project involves understanding IP routing, ICMP, and network packet manipulation.

- **Key Files**: `traceroute.py`, `util.py`
- **Spec**: https://cs168.io/proj1/

### [Project 2: Network Simulator & Routing](proj2/)
A comprehensive network simulation framework implementing routing algorithms and network behavior. Includes:
- **Network Simulator**: Core simulation engine for testing routing protocols
- **Network Visualization (NetVis)**: Interactive visualization of network topology and packet flow using Processing
- **Routing Components**: Implementation of various routing protocols

- **Key Directories**: `simulator/`, `netvis/`
- **Spec**: https://cs168.io/proj2/

### [Project 3: Transport & SDN](proj3/)
Exploration of transport layer protocols and Software-Defined Networking (SDN) using **POX**, a Python-based OpenFlow controller framework.

- **Framework**: POX network controller
- **Spec**: https://cs168.io/proj3/
- **POX Repo**: https://github.com/noxrepo/pox
